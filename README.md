OpenFace is a facial behavior analysis tool a tool intended for computer vision and machine learning researchers, affective computing community and people interested in building interactive applications based on facial behavior analysis.
Using the facial Action Units we can extract the OCEAN traits from the user for a session and give a hire-likert score where the scale is ranging from (1-7). In the end of the session the user feedback summary is displayed which can be helpful for further analysis and improvements.
